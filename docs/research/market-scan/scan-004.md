# Scan 004 — AI generators + CMS posture (re-run) + vertical-specific shops

**Date:** 2026-06-17 · **Method:** deep-research harness, 6 angles → 30 sources → 137 claims → 25 verified. **24 confirmed, 1 killed.** (Completes the cut-off scan-003 angle + adds vertical-niche shops.)

> **Headlines:** (1) **A client self-edit CMS is the NORM, not rare** among done-for-you shops — every one verified ships an editor. So **"we give you a CMS" is NOT a differentiator.** What's genuinely uncommon is our combo: **one-time fee + full code ownership + no maintenance contract.** (2) AI generators split: dev-grade (Lovable) gives code ownership + GitHub export + no lock-in; platform builders (10Web) force bundled hosting. (3) **Vertical-specific shops exist for nearly every industry we target — but they're almost all subscription/hybrid.** No vertical competitor on a true one-time/client-owns model was found.

---

## CMS / ownership reality (the key correction) · `high`

**Self-edit CMS is common, bundled inside subscriptions with no code export:**
- **UENI** — "Easy-to-use editor" on all plans. Model: **$24.99/mo + $79 setup** (list $599). Recurring.
- **Jottful** — "Magical Editor" (click-and-type text/images) on all plans. **$99 one-time build (48-hr promo; standard ~$499) + $65/$79/$169/mo.** Explicitly *"not possible to move a website from one platform to another"* — only text/images returned on exit, **no code export.**
- **My PT Website** (fitness) — drag-and-drop designer. **$499 one-time + $49/$99/$199/mo;** *"we host it, secure it, watch it 24/7"* = hosting lock-in.
- **Flavor Plate** (restaurants) — self-service menu/content CMS. **$49/mo** (+ optional $499 setup).
- **Salon SEO Websites** (salons/barbers) — "Make Your Own Updates" login *or* "We Make Updates for You." **$499 setup + $49/mo.**

→ **None offers pure one-time + full ownership.** Self-editing is table stakes; **ownership + one-time pricing is the wedge** (alongside design quality). *(This corrects scan-002's tentative "self-edit CMS = our moat.")*
- ⚠️ Refuted (0-3): "Jottful is pure subscription, no one-time fee" — confirms Jottful **does** charge a one-time build fee on top of the sub.

## AI website generators · `high`

- **Lovable** — dev-grade: *"all projects can be continuously synced to GitHub,"* output is standard Vite+React deployable anywhere or self-hosted, *"you are never locked in."* ⚠️ Caveat: Supabase backend not portable out-of-box; editor itself can't be self-hosted.
- **10Web AI** — subscription only (**$10 AI Starter → $23 AI Ultimate /mo**, $112/mo Agency Core, Agency Pro custom "as low as $2.5/site"). Mandatory bundled WordPress hosting on Google Cloud; help center: AI output *"exclusively supported on 10Web hosting, will not work on other platforms"* = lock-in (technical escape exists via raw WP migration, but the product is designed for lock-in). Cancel anytime, 30-day refund (voids once free domain activates).
- *(Wix AI, Durable, Hostinger AI, Framer, v0 did not surface verified claims this run — still open; Framer hosting-lock-in + export tools noted in sources.)*

## Vertical-specific / niche productized shops · `high`

Named players exist for nearly every target industry — **overwhelmingly subscription/hybrid:**
- **Fitness** → My PT Website ($499 + $49–199/mo)
- **Restaurants** → Flavor Plate ($49/mo)
- **Salons/Barbershops** → Salon SEO Websites (Dallas/Fort Worth+, $499 + $49/mo)
- **Home-service trades (HVAC/plumbing/roofing/solar)** → **RYNO Strategic Solutions / Blue Corona** (merged, PE-owned under EverService/Sunstone) and **Footbridge Media** ($249/mo, no setup, month-to-month)

**Ownership exceptions worth noting:**
- **RYNO/Blue Corona** — explicitly *"You own 100% of everything we create — code, content, design,"* and clients *"can take everything with them"* on cancellation. **BUT** it's an ongoing marketing subscription bundling hosting/security/webmaster support — client can't own the CMS/hosting. Adjacent (marketing agency), not like-for-like.
- **Footbridge Media** — client-friendly: **$249/mo, no setup, month-to-month, 90-day money-back, "your site goes with you."** Still recurring, not one-time.
