# Market Scan — Competitor Research Log

**Purpose:** An append-only corpus of verified competitor profiles + market insights for Studio0rbit (productized local small-business websites). Gathered on a recurring research loop. **We are gathering, not synthesizing yet** — a mass review comes later.

**How this works:** Each loop run picks NEW competitors/angles from the queue below (never repeat a covered one), runs deep research, and writes a numbered scan file (`scan-NNN.md`) with verified, cited findings. Then it updates the "Covered" list and "Queue" here so the next run (which may start with fresh context) knows what's done.

**Sibling docs:** strategic analysis lives in [`../2026-06-16-competitive-landscape.md`](../2026-06-16-competitive-landscape.md); raw links in [`../../references/links.md`](../../references/links.md).

---

## Run protocol (for each loop iteration)
1. Read this INDEX → pick the next 1–2 **uncovered** angles from the Queue (or invent a fresh one not in Covered).
2. Run the `deep-research` workflow on that angle.
3. Write findings to `scan-NNN.md` (next number) — verified claims only, with confidence + sources. Note what was rate-limited/unverified.
4. Update **Covered** (add competitors/angles done) and **Queue** (remove done, add new leads discovered) below, and add a line to the **Run log**.
5. Do NOT synthesize across runs yet — just accumulate.
