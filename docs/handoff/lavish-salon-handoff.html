How messages reach you

The enquiry form runs on Web3Forms — a lightweight form service that needs no server and no GitHub. Right now it delivers to our Studio0rbit inbox so it's tested and working from day one. When you're ready to go live, we swap in your own free Web3Forms key (a 2-minute change) so every enquiry lands straight in your salon's email. Editing The Site Today

No content system yet — edits are made in code

This build ships without a CMS, which keeps it lean and fast. Today, changes to text, images, hours, or services are quick code edits we make for you on request. If you'd like to edit the site yourself — change copy, swap photos, update hours from your phone — that's our Storyblok CMS add-on: we wire in a simple visual editor where you click a block, type, and hit Publish. Just say the word and we'll quote it.

Either way, the site is engineered so the content layer can be added later without a rebuild.
