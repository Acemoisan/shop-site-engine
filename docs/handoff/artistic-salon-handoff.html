Editing Your Content

No CMS yet — edits are in code today

To keep this build lean and fast, the site's text, prices and photos currently live in the site's code rather than a content editor. That means changes (a new price, a swapped photo, new hours) are quick edits we make for you on request — there's no dashboard to log into for content right now.

If you'd like to self-edit, we offer a Storyblok CMS add-on (a friendly point-and-click editor for text and images, with a Publish button and version history). It's an optional paid add-on, not part of this one-time build — just say the word and we'll wire it in.
